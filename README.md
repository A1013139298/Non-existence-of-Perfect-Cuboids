## 📘 What This Application Does

This application provides a formal proof that perfect cuboids do not exist. A perfect cuboid is a special shape where all edges, face diagonals, and the main diagonal have whole number lengths. The program uses deep math concepts like Mordell-Weil rank exhaustion and minimal polynomial irreducibility to prove this.

This proof involves complex mathematical tools in algebraic geometry and number theory. The application makes these concepts more formal and clear using computational checking and verification.

You don't need to understand all the math behind it to run the software. It is designed to work on your Windows computer and check the proof processes step by step.
